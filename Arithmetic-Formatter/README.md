# Arithmetic Formatter

This project takes a list of arithmetic problems and arranges them vertically.

## Function
`arithmetic_arranger(problems, show_answers=False)`

## Features
- Handles addition and subtraction
- Formats problems side-by-side
- Optional: Shows answers with `show_answers=True`
- Includes error handling for invalid input

## Example
`arithmetic_arranger(["32 + 8", "1 - 3801"], True)`
