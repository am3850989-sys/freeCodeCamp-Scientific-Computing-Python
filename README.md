# freeCodeCamp Scientific Computing with Python & Data Structures

This repository contains all 8 projects required for the freeCodeCamp certifications.

## Projects Included

### Scientific Computing with Python
1. **Arithmetic Formatter** - Formats arithmetic problems vertically
2. **Time Calculator** - Adds time and handles days later
3. **Budget App** - Category budgeting with spending chart
4. **Polygon Area Calculator** - Rectangle and Square classes
5. **Probability Calculator** - Hat and experiment simulation

### Data Structures and Algorithms
6. **User Configuration Manager** - Manage user settings with CRUD operations
7. **Hash Table** - Custom hash table implementation
8. **Tower of Hanoi** - Recursive algorithm implementation

## How to Run
Each project is in its own folder. Run the .py file to test the functions.
