# freeCodeCamp Scientific Computing with Python

This repository contains all 5 projects required for the freeCodeCamp Scientific Computing with Python Certification.

## Projects Included

1. **Arithmetic Formatter** - Formats arithmetic problems vertically
2. **Time Calculator** - Adds time and handles days later
3. **Budget App** - Category budgeting with spending chart
4. **Polygon Area Calculator** - Rectangle and Square classes
5. **Probability Calculator** - Hat and experiment simulation

## How to Run
Each project is in its own folder. Run the .py file to test the functions.
